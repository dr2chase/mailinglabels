Mailing Label Maker

This little web application renders addresses into a PDF that is formatted for Avery 5160 labels.

To use it, upload a text file containing a list of addresses, one address per line.
Each address must consist of three parts with parts separated by the '$' character.

For example, here is a sample address file:

Google$1600 Amphitheater Parkway$Mountain View, CA 94943
Apple Inc.$1 Infinite Loop$Cupertino, CA 95014
Department of Mathematics$970 Evans Hall$Berkeley, CA 94720

Uploaded addresses are not logged or archived.
Source code for this site is available on GitHub.
